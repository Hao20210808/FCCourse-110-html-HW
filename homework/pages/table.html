Structure Part 2: Table

table in HTML

When we write HTML, we will use the layout of the table editing layout, which contains several important tags, namely table, tr and td, and then combine them as a complete table



Three most common and definitely used tags in HTML tables

<table> is used to wrap the structure and content of the entire table; <tr> (table row) is used to define several rows (rows) in the table; <td> (table data) is used to define the table There are several straight lines (cloumn), as for <td>, the actual cell information is placed in it.


In addition to writing a simple table, we can also set the width, height or border style of the table. To do these settings, you must first understand several elements that can control the table, such as width, border, background, etc. .


  • width: controls the width of the table.
  • border: controls the thickness of the table border.
  • background: controls the background image.
  • colspan: controls how many fiels the cell spans.
  • rowspan: controls how many columns the cell spans vertically.



The first is the configuration of the fields

In our example, we use the <table> and </table> tags to wrap the content, the <tr>></tr> tags represent a row, and the <td>></td> tags represent a column.


If the border needs to be clearly displayed, we can add the setting of border="1" to the table, which means that the border thickness of the table is 1. In actual use, you can also set it to other values, such as HTML in general. The default border="0" will not display the border.
