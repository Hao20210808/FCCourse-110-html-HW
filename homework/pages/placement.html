The position Property

The position Property is use element to specifies the position.

There are five different types:

  • static: controls the width of the table.
  • relative: controls the width of the table.
  • fixed: controls the width of the table.
  • absolute: controls the width of the table.
  • sticky: controls the width of the table.

This element is ues top,right,bottom, and left to determine the local position.
The position neet to set first or the properties won't work.

static

This element is not affected by top, bottom, left, and right.

border: 3px solid #73AD21; }
/style

fixed element will have a viewpoet stays in the page, it won't leave even the page is scrolled. Use top, right, bottom, and left to position the element.
