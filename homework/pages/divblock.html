Structure Part 1: The Div Block

What is div tag?

HTML div tag is an important basic element in web pages. Through DIV tags, various blocks can be created in web pages, and smaller blocks can be added to each block. In addition, HTML div blocks can also be used for typesetting when designing web pages. For web browsers, each group of HTML div tags is a block-level element, which basically takes up a whole line of space, and we can also This part is adjusted through the attribute setting of div, but the current div attribute setting is roughly designed with CSS.



Basic syntax of div tag block

<div id="Customize" class="Customize" style="Customize">block content</div>


A set of standard div blocks is composed of an opening <div> tag and a closing </div> tag, and the backpacked part is the block content, which can be text, pictures, videos, frames, etc. Web page content, the id, class can be set in the <div> block at the beginning, or a style design can be started directly, that is, the style part.
