Structure Part 4: Meta

Introduction

Metadata is the data information of data. The <meta> tag provides metadata for the HTML document.

Metadata is not displayed on the client side, but is parsed by the browser. The meta element is often used to specify the description of the web page, keywords, the last modification time of the file, the author and other metadata.

Metadata can be invoked using browsers (how to display content or reload pages), search engines (keywords), or other web services.

<meta> tags always go inside the <head> element, and are typically used to specify character set, page description, keywords, author of the document, and viewport settings.

Example

                    <head>

                        <meta charset="UTF-8">

                        <meta name="description" content="HTML/CSS Web tutorials">

                        <meta name="keywords" content="HTML, CSS">

                        <meta name="author" content="簡宏宇">

                        <meta name="viewport" content="width=device-width, initial-scale=1.0">

Attribute-charset

<meta> charset is used to specify what encoding character set that the webpage content is used for. Most webpage encodings are now UTF-8. 

                    <meta charset="utf-8">

Attribute-http-equiv

  • Content-Security-Policy :

    • Content-security-policy is use to specifies a content policy for the document.

                                <meta http-equiv="content-security-policy" content="default-src 'self'">

  • Content-Type :

    • Content-type is use to specifies the character encoding for the document.

                                <meta http-equiv="content-type" content="text/html; charset=UTF-8">

  • Default-Style :

    • Default-style is use to specified the preferred style sheet to use.

                                <meta http-equiv="default-style" content="the document's preferred stylesheet">

  • Refresh :

    • Defines a time interval for the document to refresh itself.

                                <meta http-equiv="refresh" content="60">

Attribute-name

  • Application-Name :

    • Application-name is use to specifies the name of the Web application that the page represents


  • Author :

    • Author is use to specifies the name of the author of the document.

                                <meta name="author" content="Hactor Jian">

  • Description :

    • Description is use to specifies a description of the page. Search engines can pick up this description to show with the results of searches.

                                <meta name="description" content="Free web tutorials">

  • Generator :

    • Generator is use to specifies one of the software packages used to generate the document (not used on hand-authored pages). 

                                <meta name="generator" content="FrontPage 4.0">

  • Keywords :

    • Keywords are use to specifies a comma-separated list of keywords - relevant to the page (Informs search engines what the page is about).

                                <meta name="keywords" content="HTML, CSS">

  • Viewport :

    • Controls the viewport (the user's visible area of a web page).

                                <meta name="viewport" content="width=device-width, initial-scale=1.0">
